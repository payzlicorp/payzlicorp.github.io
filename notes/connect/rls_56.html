Included in Release

We are excited to announce the latest update to our software. This release includes several important bug fixes, enhancements, and new features to improve your experience.

New Features

### MCC Code Change Permission: Added permissions for changing MCC codes. ### Repay Cost and Fees: Updated the repay cost and other fees for better accuracy.

Enhancements

### Support Categories: Adjusted support categories and subcategories to meet new requirements. ### Support Case Initial Mail: Modified the initial email content for support cases to provide clearer information. ### Restricted Assign to User Functionality: This feature is now limited to internal Payzli employees for support cases.

Security Updates

### Implemented various security updates to ensure the safety and integrity of our system.

Bug Fixes

### Miscellaneous Bug Fixes: Addressed various minor bugs to enhance overall stability. ### Merchant Profile: Fixed issues related to merchant profile management. ### Account Deactivation: Resolved the problem causing accounts to get deactivated unexpectedly.

We hope these updates enhance your experience. Thank you for your continued support!
