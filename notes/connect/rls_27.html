Included in Release

This release encompasses several enhancements to the Merchant Monetary Screens, providing a comprehensive view of transactional activities for each merchant. Key features include:

  • Transactions & Activities Screen : Detailed transactional activity for each merchant is now accessible from their profile.
  • Settled Transactions - A consolidated view of settled transactions for the merchant, updated until the previous day.
  • Deposits & Withdrawals: ACH records for the merchant are now available, with a breakdown for NetAch Merchants.

  • Chargebacks: Access chargeback records conveniently from the merchant’s profile.

  • Moved to new cron Job mechanism

  • Tsys Authorisations would be added soon to the reports and few UI updates in the next release.
