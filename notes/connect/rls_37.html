Included in Release

This release is packed with improvements and fixes to enhance overall experience and ensure better stability.

Partner On Boarding Refactor

Introducing a streamlined “Partner Flow” within Connect to efficiently onboard new partners and manage their applications through various stages.

Esquire Cron Job

  • Bank jobs run hourly.
  • Along with main file it will upload merchant application and attachments.
  • Bug fixes - updated line encoding, handled null values -

User Type (Merchant/Partner) In Organization List

This feature will help to differentiate between Merchant and Partner names.

Merchant Application Signing Flow

  • DBA Name should be a required field
  • Ownership percentage of authorized rep should be a required field
  • Tax Id should be a required field

Removed ACH Reject Fee from flex fees

Partner Profile Update HotFix - remove extra validation

All the partner information should be editable in any state with Edit Partner Profile permission. Removed extra validation on first name, last name, phone number , company name.

Partner Duplication Bug

Patch For Merchant Account Overlap

Restrict multiple opening tabs to avoid merchant account overlap bug.

Risk-UI Issues

These updates and fixes aim to enhance functionality, improve user experience, and address various issues encountered within the software. We appreciate your continued support and feedback. If you have any questions or encounter any issues, please feel free to reach out.
