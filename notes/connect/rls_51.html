Included in Release

This release includes bug fixes to enhance the stability and performance of the application.

Bug Fixes

  • No Equipment Selected Error Message in Sign-Up Flow Resolved an issue where merchant encountered a “No Equipment Selected” error message during the sign-up process. This fix ensures a smoother and more intuitive sign-up experience for new merchants.

  • Unverified User Error in Login and Account Creation Fixed a bug that caused an “Unverified User” error during login and account creation. This update allows users to log in and create accounts without unnecessary interruptions. Enhanced the clarity of error messages displayed on the user interface, providing users with more precise information to resolve issues quickly.

Impact

These fixes are aimed at easing the merchant sign-up process, ensuring a more seamless and efficient onboarding experience.
