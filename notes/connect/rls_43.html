Included in Release

This release focusses on a large number of fixes and UI enhancements across Payzli Connect modules such as support, residuals, risk and other modules.

Residuals

  • The capability to create or assign an existing residual program to the sales partner has been added. This program can be assigned to a new or existing residual hierarchy

Risk

  • The capability to add an Underwriting Summary has now been added to the Merchant profile. This summary screen contains details of the various underwriting checks that have to be entered and serve as a reference screen

Connect UI/UX

  • The Connect Top header is now revamped with a better user experience. Hitting ‘/’ on the keyboard is now a shortcut to access advanced search.

Other enhancements and Fixes:

  • Quashed a bug which was causing merchant account details to overlap between browser tabs if Connect is open in multiple tabs simultaneously
  • Fixed a bug which was causing the leads share hyperlink to get truncated
  • The date format was fixed in the Transactions and Activities section
  • Templates for SMS/Email/History logs have been revamped across Connect
  • Deleted users would now be visible in Connect

Quality Assurance:

  • The QA team worked on various automation scripts related to UI and the onboarding flow
