Included in Release

Resend Invite User Link Functionality:

  • Added the ability to resend invite links to users from the user screen.
  • Set new user status as active by default.
  • Enhances user management and communication.

Residual Hierarchy Screen

  • This will provide ability to manage hierarchy of residual programs.

Bug Fixes in Delete User Functionality:

  • Addressed issues related to deleting users.
  • Show deleted users in users screen.
  • Improves reliability and stability.

Update SMS, Email, and History Log Comments:

  • Enhanced language accuracy and user experience in Connect.
  • Users will benefit from clearer communication.

Search Functionality Issue:

  • Investigated and resolved issues related to search functionality.
  • Users can now search more effectively.

Handling No Partner Organization:

  • Partners without any organization will now appear as “First Name Last Name” in the partner selection component.

Session Caching:

  • Added session caching to improve response time.
  • Enhances overall performance.

UI/UX Enhancements - Updated Font Family:

  • Improved font family for a better user interface.
  • Aesthetic enhancements.
