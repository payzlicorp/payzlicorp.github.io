What’s is new release?

  1. Legacy Partner Adjustment: Improved adjustment functionalities for legacy partners, ensuring a seamless transition to the new system.
  2. Sort Code Updates: New sort codes have been provided to the repay system. The update is expected to be completed within 7 working days.
  3. Import Existing Partners: Introducing the ability to import existing partners, simplifying data migration and setup processes.
  4. Bank Automation: Hourly job automation to upload merchant account data recently approved but not yet sent to the bank. Ensures timely processing and reduces manual efforts.
  5. Equipment Purchase Options Adjustment: Added Pay In Full or Terminal Rental Program toggle for equipment selection. Prices and flex fees now dynamically adjust based on the selected option, providing more flexibility to users.
  6. Notifications Email Body Changes: Enhanced email notifications with updated content and formatting for improved communication.
  7. User Interface Improvements: Renamed ACH Daily Adjustment Flag to Combined ACH for clarity. Streamlined underwriting section by removing Account Declined status for a more intuitive user experience. Checked and optimized team permissions at all levels for smoother collaboration.
  8. Data Security Enhancements: Implemented validation to invalidate merchant app agreements if any data is modified, ensuring data integrity and security.
  9. Login Experience: Redesigned the Login with Payzli button for a more user-friendly login process.
  10. Improved Filters: Enhanced stability and functionality of filters in sliders, providing a more reliable filtering experience.
  11. Repay Module Enhancements: Implemented handling of extra quotes added in strings, ensuring accurate data processing from the Repay side.

Bug Fixes: 1. DBA Name Fix: Corrected an issue where the DBA name on the merchant application did not carry over correctly to MES. 2. Owner Profile Information: Resolved missing information in the owner profile section, ensuring complete and accurate data representation. 3. SonarCloud Integration: Addressed bugs related to SonarCloud integration on the frontend for improved code quality. 4. Team Management Fixes: Fixed the delay issue when adding a new team, ensuring instant availability in the administration module. Addressed pagination issues for smoother navigation across the platform. 5. Address Dropdown and UI Glitches: Cleared cached values in the address dropdown for accurate location selection. Fixed screen glitches and placeholder messages during transitions to underwriting, ensuring a seamless user experience. 6. Numeric Input Fix: Corrected the issue where numbers were added next to zero in increment text fields, providing accurate input validation. 7. Duplicate Members Issue: Resolved the problem where duplicate members were displayed while adding new members to teams.

To complete legacy partner adjustments below operations need to be handled by operation team - Invite users - Add documents - Update sort codes in repay
