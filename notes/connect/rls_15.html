Included in Release

Boarding Cost Plus Priced Accounts to MES:

  • When boarding Cost Plus priced accounts to MES in Repay, ensure ‘Enhanced Cost’ is selected as ‘No’ and ‘Cost Plus Plus’ is selected as ‘No.’

Visa Error Fix:

  • Resolved Visa Error 34000: “Cursor xvisatidsdetails does not exist.”

Statement Cron Job:

  • Implemented Statement Cron Job for automated statement generation.

DDA Change Email Notification:

  • Email notification to risk@payzli.com for DDA changes done by a merchant on any account.

User Type Change:

  • Provision to change the type of user from contact to authorized representative/owner and vice versa.

Assigned Members in Email:

  • Assigned members to a ticket will now be displayed in email notifications (excluding merchants and sales partners).

Attachment Section Enhancement:

  • Resolved link overflow issues in the attachment section for lead, underwriting, support, and partner modules.

Ticket Person Suggestion Fix:

  • Fixed the issue where a person added to the ticket was appearing in the suggestion list for leads, partners, underwriting, and support.

Owner Birthdate Update Fix:

  • Resolved the invalid format error when updating owner birthdate in Repay.

Lead Conversion Update:

  • Lead conversion now correctly sends the Merchant app link to the second owner’s email when two owners are added.

Save Equipment Info API Update:

  • Implemented sending terminal IDs in the save_equipment_info API payload for each equipment in the merchant profile screen.

Attachment Symbol Update:

  • Implemented the use of the ‘Asterisk’ symbol for the ‘Choose File’ field in the attachment section for leads, partners, and underwriting modules.

Special Characters in Comments Fix:

  • Resolved the 500 error occurring when adding special characters to the comments section.

Character Limit for Corp Name:

  • Implemented a character limit for the Corp name field to avoid affecting the Application UI.

SignalR Issue on Team Deletion Fix:

  • Resolved the SignalR issue when deleting a member from a team in Teams.

Email Format Error Fix:

  • Fixed the email format error when creating a sort code, ensuring correct inclusion of Name and Company name.

Lead Conversion Error Fix:

  • Fixed the issue in lead conversion where adding two owners sent the merchant app link to the second owner’s email.

Repay Error Resolution:

  • Resolved Repay Error - Internal Program Error.

KYC KYB Payload Discrepancy Fix:

  • Resolved the KYC KYB error where the payload expected by Lendflow differed between the sandbox and production environments.

Merchant Contact Field Update:

  • Updated “Phone” to “Mobile Number” for Merchant contact field.

These updates aim to enhance system stability, improve user experience, and address reported issues.
