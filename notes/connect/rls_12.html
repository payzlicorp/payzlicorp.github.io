Enhancements & Updates:

Merchant Leads and Contacts:

  • You can now add Merchant Contacts to merchant leads. A new contact section has been added to the ownership profile, including fields for name, email, phone number, and title. This feature is available in both the merchant profile and the Add New Lead slide-out.
  • Contacts are now accessible in email selection.
  • Contacts will no longer appear as owners in the converted app flow.
  • The alignment issue with the remove button in the merchant profile screen has been resolved.
  • The sequence for adding entities has been updated to Auth Signer, Add Owner, and Add Contact.
  • A new position field has been added for Contacts.

Partner Manager Interview Mail UI:

  • Issues with the Partner Manager interview mail UI have been resolved.

Boarding Errors - Kiesler Police Supply:

  • Credit report errors in production due to password change have been fixed.
  • DBA name is now trimmed to 30 characters in repay (tested in QA).
  • Discover MID is now correctly added in repay (tested in QA).
  • Old flex fees will no longer be restored if all flex fees are removed and saved. This issue has been fixed (tested in dev).
  • Monthly processing volume is now an optional field. It will be required only at lead creation and should be filled in at the time of conversion.

Password Reset:

  • Errors related to password reset have been resolved.

Communication Logs:

  • Logs for communication have been removed from history logs.

Support Case Notifications:

  • The notification mail header while creating a support case has been updated to “New Case in Support.”

Business Address Editing:

  • Editing the business address will no longer give an error on empty search.

Bug Fixes & Patches:

Language and User Interface:

  • The default language has been updated to English (En).

Visual Enhancements:

  • The color of attachment links has been changed for improved visibility and readability.

We hope these enhancements and bug fixes improve your experience with Connect. As always, thank you for your continued support.
