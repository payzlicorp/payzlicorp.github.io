Release Notes: Version 4.0.0

Overview

Version 4.0.0 brings a plethora of enhancements, bug fixes, and new features to improve the overall performance and usability of our POS system. This release addresses critical issues, refactors code for better maintainability, and introduces exciting new functionalities.

Highlights

  • Bug fixes: Numerous bug fixes reported by internal team.
  • Elastic Fixes: Improved Elasticsearch functionality.
  • Signal R Issues: Resolved Signal R issues.
  • Proposal and Quote Module: Introduced a new module for proposals and quotes.
  • Printer Integration : Integrated printer for printing sale receipts.
  • Users Services & Products API Enhancements: Improved APIs for user services and products
  • UI Fixes: Addressed UI issues including text overflow, alignment, and scroll problems reported.
  • Validation Changes: Improved data validation logic.
  • Cron Job and Scheduler Fixes: Fixed issues related to cron jobs and scheduler functionalities.
  • Caching: Integrated Redis, a distributed cache which improved the overall performance of the product.
  • Refactoring: Refactored codebase for better performance and maintainability.

We sincerely thank all contributors for their valuable efforts and contributions to make this release possible. Your dedication and hard work are greatly appreciated.
