Release Notes: Version 6.0.0

Overview

Version 6.0.0 brings a plethora of enhancements, bug fixes, and new features to improve the overall performance and usability of our POS system. This release addresses critical issues, refactors code for better maintainability, and introduces exciting new functionalities.

Highlights

  • Bug fixes: Numerous bug fixes reported by internal team.
  • SignUp/Login page revamped
  • We are now able to view custom dates, all employees and filter for individual employees, that allows the time card entries to be adjusted/edited.
  • Integrated Amplitude.
  • Integrated Crashlytics.
  • Vietnamese translation issues fixed.

We sincerely thank all contributors for their valuable efforts and contributions to make this release possible. Your dedication and hard work are greatly appreciated.
