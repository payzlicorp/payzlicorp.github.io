Release Notes: Version 6.0.0

Overview

Version 6.0.0 brings a plethora of enhancements, bug fixes, and new features to improve the overall performance and usability of our POS system. This release addresses critical issues, refactors code for better maintainability, and introduces exciting new functionalities.

Highlights

  • Bug fixes: Numerous bug fixes reported by internal team.
  • VISA Unified Checkout Integration
  • Allow booking for multiple appointments in single time slot, allow appointment bookings outside of working hours.
  • Amplitude integration
  • Added timecard report.
  • Email Templates updated.
