Release Notes: Version 7.0.0

Overview

Version 7.0.0 introduces a wide range of enhancements, bug fixes, and new features designed to improve the performance and usability of our POS system. This release addresses critical issues, refactors code for better maintainability, and adds exciting new functionalities.

Highlights

  • Numerous bug fixes reported by the internal team.
  • Integration with VISA Unified Checkout.
  • Ability to book multiple appointments in a single time slot and make appointment bookings outside of working hours.
  • Integration with Amplitude.
  • Addition of a timecard report.
  • Updated email templates.
